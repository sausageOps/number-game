# 🎮 Number Guessing Game (Pygame)

An interactive number guessing game built using Python and Pygame.
Originally started as a CLI-based project and later upgraded into a full GUI application with sound effects and executable support.

---

## 🚀 Features

* 🎮 Graphical User Interface (Pygame)
* 🎯 Difficulty selection (Easy / Medium / Hard)
* ⌨️ Keyboard-based input system
* 🔊 Sound effects (click & win)
* 🔁 Restart / Replay functionality
* 🎨 Visual feedback system
* 📦 Standalone `.exe` build

---

## 🛠️ Tech Stack

* Python 3.11
* Pygame
* PyInstaller

---

## ▶️ How to Run

### Run with Python

```bash
py -3.11 main.py
```

### Run Executable

Go to:

dist/main.exe

Double-click to start the game.
